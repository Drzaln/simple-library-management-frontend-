## How To Run

1. Clone this repository
   ```
   $ git clone https://github.com/Drzaln/simple-library-management-frontend-.git
   ```
2. Install all depedencies on the package.json
   ```
      cd simple-library-management-frontend-
      npm install
   ```
3. Run program
   ```
   $ npm start
   ```
